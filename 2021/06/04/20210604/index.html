随机过程之离散马可夫链笔记

Posted on 2021-06-04 | In Statistics

以下定义和性质针对于离散状态、离散时间的时齐马可夫链。

一些说明

事物所处的状态（通常用 表示）称为状态，状态的全体称为状态空间（记为）。

一些定义

马可夫链（Markov）实质上是说明未来的状态与现在的状态有关，而与过去的状态无关，用条件概率表示为

  • 一步转移概率：从状态 经过 1 次到状态 的概率，记为
  • n 步转移概率：从状态 经过 n 次到状态 的概率，记为
  • 一步转移矩阵：第 行第 列表示从状态 经过一次到状态 的概率，记

  • n 步转移矩阵：第 行第 列表示从状态 经过 n 次到状态 的概率，记

  • 一步首达概率：从状态 经过1次到状态 的概率，且中间不能经过状态 （即为首次到达） ，记为
  • n 步首达概率：从状态 经过 n 次到状态 的概率，且中间不能经过状态 ，记为
  • 状态 可达 ： ，记为
  • 状态 与 互通： ，记为
  • 类：互通的状态称为同一类，所有状态为一类则称为马可夫链为不可约的。
  • 状态的周期：称状态 的周期为经过 n 步回到状态 的所有的 n 的最大公约数，如果周期为 1 称为非周期的，周期大于 1 称为周期的，不存在这样的n 称为周期无穷大。

周期是类性质

类性质指同一类具有相同的属性，这里同一类具有相同的周期。_不同的类周期不一定不同_。

常返是类性质

相同的类具有相同的正常返、零常返、非常返

周期无穷必然自身为一类

周期无穷即不存在 n 能回到其自身状态，那不可能存在状态与其互通，即自身一类。

状态有限的马可夫链不具有零常返状态，不可能全为非常返状态

如果不可约的有限的马可夫链即可得出全部状态均为正常返状态。
